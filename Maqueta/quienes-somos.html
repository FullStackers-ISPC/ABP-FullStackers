Sobre EasyStock

EasyStock es un sistema web de gestión de inventario desarrollado como proyecto académico para optimizar el control de productos, movimientos y stock dentro de una organización.

El proyecto busca mejorar la trazabilidad, centralizar la información y facilitar la administración de recursos mediante dashboards intuitivos y módulos de gestión adaptados a distintos roles de usuario.
